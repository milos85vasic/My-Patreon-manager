Component Design Choice Details

Organization Support Changes

• GITHUB_ORGS
• GITLAB_GROUPS
• GITFLIC_ORGS
• GITVERSE_ORGS

Provider Updates:
1. GitLab: Add user repo fallback
2. GitFlic: Add /user/repos endpoint
3. GitVerse: Add /user/repos endpoint
4. All: Wildcard (*) support for org discovery
Illustration System Changes
LLMsVerifier Extension:
• Add OpenAI DALL-E provider
• Add Stability AI provider
• Add Midjourney provider
• Support both vision & creation modes

My-Patreon-Manager Integration:
• New IllustrationGenerator service
• Database schema for images
• Integration with content pipeline
• Patreon image upload support
